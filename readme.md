# Git
controle de versão simples, rápido e Open source

## Versão
git --version

## Configuração
git config --global user.name "nome"  
git config --global user.email "e@mail"  

git config user.name  
git config --list

## Comandos básicos
git status  
git add .  
git commit -m "msg"  
git log

### git log
git log --author="sam"  
git shortlog  
git log --graph  

## Outros comandos
git show  
git diff  

### git show
git show 71bc6dc7 (8 digitos)  

### git diff
git diff --name-only
